O que é Git Clone

  1. GitClone é possível clonar um repositório do GitHub.com em seu computador local para facilitar a correção de conflitos de mesclagem, adicionar ou remover arquivos e enviar commits maiores. Ao clonar um repositório, ele é copiado do GitHub.com para seu computador local.

  1. Copie a URL do repositório.

    • Para clonar o repositório usando uma chave SSH, incluindo um certificado emitido pela autoridade de certificação SSH da sua organização, clique em SSH e em .
    • Para clonar um repositório usando GitHub CLI, clique em GitHub CLI e selecione.
